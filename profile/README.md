# Tooark

Website: https://tooark.com/  
GitHub: https://github.com/Tooark

## Sobre

**Tooark** é uma organização dedicada a projetos de software e ferramentas web mantidos por Paulo Freitas e colaboradores. Nosso foco inclui extensões, bibliotecas e utilitários que facilitam desenvolvimento, automação e experiências web modernas.

**Missão:** criar e manter projetos úteis, bem documentados e fáceis de contribuir.
**Visão:** compartilhar soluções abertas que acelerem o trabalho de desenvolvedores e equipes.

## Projetos Principais

### Extensões e Plugins

Repositórios com extensões para editores e plataformas (ex.: publicações no Open VSX).

### Bibliotecas e Utilitários

Código reutilizável para automação, integração e componentes web.

### Sites e Portfólios

Projetos de front end e sites pessoais, incluindo o portfólio de Paulo.

## Como Contribuir

### Abrir Issues

Use issues para reportar bugs, sugerir melhorias ou pedir suporte. Seja claro no título e descreva passos para reproduzir o problema.

### Pull Requests

1. Fork do repositório.

2. Crie uma branch com nome descritivo (feature/nome, fix/descricao).

3. Adicione testes e documentação quando aplicável.

4. Abra PR com descrição do que foi alterado e por quê.

### Padrões de Código

Siga o estilo do repositório; inclua lint e formatação quando necessário.

### Comunicação

Use comentários em PRs e issues para discutir mudanças. Mantenha o tom colaborativo e objetivo.

### Licença e Governança

Licença padrão: Apache 2.0 (ou outra licença escolhida por repositório).  
Governança: projetos podem ter mantenedores específicos; verifique o arquivo MAINTAINERS ou a seção de colaboradores em cada repositório.
